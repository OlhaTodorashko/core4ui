# pnbi-base

> Authentication, Helpers and Components for CORE4 webapps based on VueJS

## Installation

```
Add this toy your dependencies:
  "dependencies": {
    "core4ui": "https://github.com/plan-net/core4ui.git#develop",
```
#### package.json

```bash
"core4ui": "https://github.com/plan-net/core4ui.git#develop"
```
